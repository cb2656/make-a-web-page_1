Lesson 3: Adding CSS Style and HTML Structure

CSS stands for Cascading Style Sheets. An Important idea is known as:
Inheritance and understanding it requires abstract thinking.

There are many selections for CSS Selectors; using the Mozilla CSS Reference.
--H1 is a selector and by using curly braces after and at the bottom, everything inbetween is the "rule."
The line of code that says background-color : black; is a declaration. background-color is a property and black is the value of that property.

Since everything on the web is in boxes, structure within the boxes is key. Elements within elements creates a structured 'box-like' model of a web page. The "Divs" and "Spans" help to build the structure also.

Avoiding repetition is very important. By copying and pasting the same code with minor diferences you open yourself up to potential Errors.

Box Sizing:
There are four main points about box sizing.
--HTML elements are boxes and each box has 4 components.
--There are so many components to each box, it can be hard to get the size of a box right.
--There are two techniques to help deal with sizing issues.
--Different browsers work differently. So,different browsers can display the same code differently.

Box positioning:
Adding the rule "display: flex; to the right CSS will let the "divs" appear next to each other.
